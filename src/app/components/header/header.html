About Me :
 This project is a user management dashboard built with Angular, demonstrating core concepts for students during a training session.
It fetches user data from the DummyJSON API using the Fetch API and displays it in an interactive UI.

Key Features & Concepts Covered:
1. Data Fetching – Uses fetch() to GET user data from DummyJSON
2. Component Communication – Input() & Output() for parent-child interaction
3. Template Directives – looping, conditional rendering
4. Data Binding – Interpolation, property binding, event binding
5. Dynamic Styling – Conditional classes
6. Event Handling – Click events, custom event emission
7. TypeScript – Strong typing for API responses & component logic
8. Responsive UI – Styled with CSS for a clean, modern layout
